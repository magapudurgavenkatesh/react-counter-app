# React Counter App

A simple counter app built to understand how state works in React using useState.

## Features
- Increment / Decrement functionality
- Dynamic step control
- Min/Max limits with UI feedback
- State persistence using localStorage

## What I Learned
- How useState manages state across re-renders
- Why functional updates (prev => ...) are important
- How React updates the UI when state changes
